Learn the Fundamentals

We teach you the skills you need to build a creative and responsive website.

In this course, you will learn the basics of building a website from scratch using HyperText Markup Language (HTML) and Cascading Style Sheets (CSS). We will move on to learning JavaScript as well. These are the building blocks of web development.

Learn React

Dive in and learn React.js from start to finish! Learn Reactjs, Hooks, Redux, React Routing, Animations, Next.js and more!

You will develop a strong understanding of React's most essential concepts: JSX, class and function components, props, state, lifecycle methods, and hooks. You will be able to combine these ideas in React's modular programming style.
